# People's Priority Portal

People's Priority Portal is a simple web application that helps citizens report local issues and vote on the problems that need the most attention. It provides a single platform where issues can be organized and prioritized based on public votes.

The project was built for a hackathon using **HTML, CSS, JavaScript, and LocalStorage** without any backend or external frameworks.

## Features

- Add new public issues
- Vote on existing issues
- Delete issues with confirmation
- Search and filter issues
- Sort by votes, priority, or newest
- Live statistics dashboard
- Dark mode support
- Responsive design
- Data stored using LocalStorage

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- LocalStorage API

## Running the Project

1. Download or clone the project.
2. Open `index.html` in any modern browser.
3. Start adding and voting on issues.

## Limitations

- Data is stored only in the browser.
- No user authentication.
- No backend or shared database.
